# Deploy Cloudflare Pages

Эта папка уже подготовлена для публикации сайта.

Что внутри:
- `index.html` — главная страница сайта
- `main_portfolio.css` — стили
- `main_portfolio.js` — логика и модалки
- `*.mp4` — видео для кейсов
- `*.png` — фото профиля

Как опубликовать бесплатно:
1. Откройте Cloudflare Pages.
2. Выберите `Direct Upload`.
3. Загрузите содержимое этой папки или ZIP-архив `deploy_cloudflare.zip`.
4. После публикации получите ссылку и сможете отправлять её клиентам.

Важно:
- В публикацию не нужно загружать всю рабочую папку проекта.
- Для сайта нужен именно этот подготовленный набор файлов.
